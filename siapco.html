MOBILE APP HRM SIAPCO

Descripción: Sistema de administración de recursos humanos diseñado para permitir a los empleados de SIAPCO subir evidencias fotográficas de sus actividades diarias. Además, ofrece la posibilidad de ver, firmar y descargar sus recibos de nómina de manera digital de forma quincenal. La aplicación móvil está desarrollada con React Native y TypeScript, y se comunica con una API REST completamente implementada en PHP, adaptada a las necesidades específicas de las funcionalidades de la app. Además, la aplicación está integrada con la API de Google, donde se almacenan los correos electrónicos de los usuarios con acceso a la misma. La seguridad y privacidad de la información de cada usuario son prioridad, implementándose medidas estrictas para proteger sus datos personales y garantizar su confidencialidad en todo momento.
